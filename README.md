Projects
01 · NGX Multi-Asset Risk Intelligence Dashboard
Tools: Excel · Power BI · Python
Domain: Portfolio Risk | Investment Analysis | Stress Testing
A quantitative risk model built on Nigerian Exchange Group (NGX) market data covering January 2023 to May 2026. The model analyses a three-stock equal-weight portfolio — GTCo, Dangote Cement, and Seplat Energy — across five risk dimensions and delivers an interactive Power BI dashboard for portfolio monitoring.
What was built:

Daily return calculations across 890 trading days
Value at Risk (VaR) using both Parametric and Historical methods at 95% and 99% confidence
Sharpe Ratio, Max Drawdown, Beta, and Annualised Volatility per asset
Composite Risk Score (0 to 100) combining four weighted risk factors
Return Correlation Matrix across five assets including USD/NGN FX exposure
Macro stress testing across five scenarios: NGN devaluation, oil price crash, CBN rate hike, market crash, and bull run
Interactive Power BI dashboard with KPI cards, price history, rolling volatility, risk ranking, and scenario P&L visuals

Key Findings:

Equal-weight portfolio delivers 74.07% annualised return with a Sharpe Ratio of 1.56 
Worst single-day loss at 99% confidence (Historical VaR): 4.43%
Under a full market crash scenario, portfolio loses 35% — equivalent to NGN 350,000 on every NGN 1,000,000 invested
GTCo carries the highest composite risk score (69/100); USD/NGN the lowest (14/100)
Near-zero correlations between assets confirm meaningful diversification benefit
